## Description

用 Nest.js 框架实现邮件验证码注册功能

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Attention!!

```bash
# use mailer
请在根目录 .env 文件替换邮件服务配置
```
